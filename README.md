# My Healthy Calculator (Java)

A simple Java console application that calculates:

- Body Mass Index (BMI)
- Body Adiposity Index (BAI)

## Features
- Menu selection (BMI / BAI)
- User input handling
- Health category classification
- Loop for repeated calculations

## How to Run

1. Compile:
   javac MyHealthyCalculator.java

2. Run:
   java MyHealthyCalculator
